# Simple private Chat Application with Socket.io

##  prerequisite
      ![Node.js](https://nodejs.org/en/)
      ![Angular9](https://angular.io/)
     
## Client
         
         To run the client    
	         1. npm i
	         2. npm start or ng serve 


# Service
        To run the service
			1. npm i
			2. npm run dev - developement
			3. npm run prod - production
